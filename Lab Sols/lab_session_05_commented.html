def gen_d_regression_samples(n_samples=50, p=3, reg_type='linear'):
    
    if reg_type=='linear':
        np.random.seed(seed=1) 
        x = np.random.uniform(low=0., high=1.5, size=(n_samples, p))
        y = np.sum(x, axis=1) + np.random.normal(scale=0.1, size=n_samples)
        df = pd.DataFrame(x, columns=['x' + str(i) for i in range(p)])
        df['y'] = y
    else:
        np.random.seed(seed=1) 
        x = np.random.uniform(low=0., high=1.5, size=(n_samples, p))
        y = np.sum(x, axis=1) + np.sum(x**2, axis=1) + np.random.normal(scale=0.1, size=n_samples)
        df = pd.DataFrame(x, columns=['x' + str(i) for i in range(p)])
        df['y'] = y

    return df

Exercise 1: $k$-nearest neighbors classification¶

We briefly recall the $k$-nearest neighbors ($k$-NN) algorithm seen in lab session 2, which is a classification method.

Consider a dataset $\mathcal{D} = (X, y)$ where the $k$-th coordinate of the $i$-th observation is denoted by $x^{(i)}_k$ and the corresponding label is $y^{(i)}$, or equilvalently: $$\boldsymbol{X} = \begin{pmatrix} x_1^{(1)} & \dots & x_p^{(1)} \\ \vdots & \dots & \vdots \\ x_1^{(n)} & \dots & x_p^{(n)} \end{pmatrix} \quad \quad \boldsymbol{y} = \begin{pmatrix} y^{(1)} \\ \vdots \\ y^{(n)} \end{pmatrix}.$$

Fix an integer $k$. Given a new observation $x \in \mathbb{R}^p$, we predict its class $\hat{y}$ in the following way.

Let $x^{(i_1)}, \dots, x^{(i_k)}$ be the $k$ observations in $X$ that minimize the Euclidean distance $\|x - x^{(i)}\|_2$. We choose $\hat{y}$ that is the predominant occurence among the corresponding classes $\{ y^{(i_1)}, \dots, y^{(i_k)} \}$. In case of a tie, i.e., two (or more) classes are the most frequent, $\hat{y}$ is chosen randomly (uniformly) among these classes.

def oneNN(X_train, y_train, x):
    """
        param X_train: np.array, shape N x p
        param y_train: np.array, shape N
        param x:       np.array, shape p
        return: y_hat, class prediction.
    """
    ### BEGIN SOLUTION
    # First, we construct a list of distances from x to each data point in X_train.
    # Each distance is computed using np.linalg.norm(x - x_train).
    # Then, we predict y_hat label by choosing the label of the closest point in X_train (1-NN).
    # The smallest distance is the min in the list. However, we don't need the distance itself but its position.
    # We use np.argmin instead, and finally we look up the corresponding label in y_train and return it.
    y_hat = y_train[np.argmin([np.linalg.norm(x - x_train) for x_train in X_train.to_numpy()])]
    return y_hat
    ### END SOLUTION

Exercise 2: Linear regression and its extensions¶

We briefly recall the framework of linear regression.

Consider observations $x = x^{(1)} \dots x^{(n)}$, with $x^{(i)} \in \mathbb{R}^p$, and labels $y^{(1)} \dots y^{(n)}$. Given an observation $x^{(i)}$, and a vector $\theta = (\theta_0 \dots \theta_p)^T$, we produce an estimation $\hat{y}^{(i)}$ of $y^{(i)}$ of the following form: $$ \hat{y}^{(i)} = \theta_0 + \theta_1 x^{(i)}_1 + \dots + \theta_p x^{(i)}_p.$$

def pred(x, theta):
    """
        param x:     np.array of shape p
        param theta: np.array of shape (p+1)
    """
    ### BEGIN SOLUTION
    # First, we add a bias term (IMPORTANT!) to our data point x, a p-dimensional vector, which also makes its shape p+1.
    # Second, we compute a dot product of the extended x vector with the theta vector, the weights.
    # We return the prediction given by this regression model.
    return np.dot(np.concatenate((np.array([1]), x), 0), theta)
    ### END SOLUTION

We gather all samples in matrices $\boldsymbol{X}$ and $\boldsymbol{y}$, where $$ \boldsymbol{X} = \begin{pmatrix} x_1^{(1)} & \dots & x_p^{(1)} \\ \vdots & \vdots & \vdots \\ x_1^{(n)} & \dots & x_p^{(n)} \end{pmatrix} \quad \text{and} \quad \boldsymbol{y} = \begin{pmatrix} y^{(1)} \\ \vdots \\ y^{(n)} \end{pmatrix}. $$

We propose a loss function $E(\theta) := \| y - X^T \theta\|^2_2$ which is the sum of squared errors (differences between the true and predicted value for each $x$ in $X$).

def E(X, y, theta):
    """
        :param X:     np.array of shape N x p
        :param y:     np.array of shape N
        :param theta: np.array of shape (p+1)
        :return:      float, evaluation of the error function.
    """
    ### BEGIN SOLUTION
    # First, we construct a list of predictions by calling pred function on the x data point with theta weights vector.
    # Then, we subtract y, the correct output vector, and compute the distance using np.linalg.norm
    # We return squared distance.
    return np.linalg.norm([pred(x, theta) for x in X] - y) ** 2
    ### END SOLUTION

We want to find an optimal vector $\theta^*$ so that the loss made by using $\hat{y}^{(i)}$ to approximate $y^{(i)}$ is small. With vector notations, it reads: $$\theta^* \in \mathrm{argmin} (E(\theta)).$$

Since $E(\theta)$ is convex, $\theta^\star$ exists and is unique.

Recall that to find $\theta^\star$, a straightforward solution is to differentiate $E(\theta)$ so as to obtain its gradient, set it to 0 and solve the resulting equation.

This yields a closed form, namely the optimal $\theta^\star$ is given by: $$ \theta^\star = (X^T X)^{-1} X^T y$$

  • You can use np.dot(A,B), or equivalently A.dot(B) to compute the matrix-matrix (or matrix-vector) $A \cdot B$ product between two numpy arrays A and B. Beware of the shapes!

def linreg(X, y):
    """
        param X: np.array of shape N x (p+1)
        param y: np.array of shape N
        return:  np.array of shape (p+1)
    """
    ### BEGIN SOLUTION
    # First, we compute X^T*X
    # Second, the inverse: (X^T*X)^(-1)
    # Third, X^T*y
    # Finally, we return (X^T*X)^(-1) * X^T*y
    XX = np.dot(X.T, X)
    invXX = np.linalg.inv(XX)
    Xy = np.dot(X.T, y)
    return np.dot(invXX, Xy)
    ### END SOLUTION

Polynomial regression¶

We consider the specific case where the observations $x^{(i)}$ are real-valued (that is $p=1$). For an integer $k$ fixed, and a vector $\theta = (\theta_0 \dots \theta_k)^T$, we propose to estimate $y^{(i)}$ in the following way: $$ \hat{y}^{(i)} = \theta_0 + \theta_1 x^{(i)} + \theta_2 (x^{(i)})^2 + \dots + \theta_k (x^{(i)})^k. $$ Here, $(x^{(i)})^j$ means the $j$-th power of $x^{(i)}$.

As in standard linear regression, the goal is to find a $\theta^*$ so that $$ \sum_{i=1}^n (y^{(i)} - \hat{y}^{(i)})^2 $$ is minimal.

def polynomial_expand(x, k):
    """
        param x: np.array of shape N
        param k: integer
        return:  np.array of shape N x (k+1)
    """
    ### BEGIN SOLUTION
    # First, we create a list of powers of x from 0 up to k+1 (without k+1).
    # The list has k+1 elements, each of which is an N-dimensional vector.
    # x ** 0 gives us the vector of 1s.
    # Then, we convert it to a numpy array.
    # We transpose it, because the array we obtained from the list has (k+1) rows and N columns.
    return np.array([x ** i for i in range(k+1)]).T
    ### END SOLUTION

Exercise 3: logistic regression¶

The logistic regression is a common model used to perform binary classification that is actually formulated as a regression problem.

We have a learning set $\mathcal{D} = \{\boldsymbol{x^{(i)}}, y^{(i)}\}_{1 \leq i \leq n}$, where $y_i \in \{0,1\}$ and $x_i \in \mathbb{R}^p$.

We consider the logistic function: $$ \sigma : t \mapsto \frac{1}{1 + e^{-t}}.$$ Note that $\sigma$ takes values in $(0,1)$ and will be used to estimate $y^{(i)}$ given $x^{(i)}$. More precisely, given a weight vector $\theta = ( \theta_0 \dots \theta_p )^T$ and a vector $X^{(i)} = (1, x^{(i)}_1 \dots x^{(i)}_p)$, we make the following estimation: $$ \hat{y}^{(i)} = \begin{cases} 1 \text{ if } \sigma(X^{(i)T} \theta) > 1/2 \\ 0 \text{ otherwise } \end{cases} $$

We want to find a good $\theta$. One can show that the optimal $\boldsymbol{\theta}^* = \begin{pmatrix} \theta_0^* & \dots & \theta_p^* \end{pmatrix}^T$ minimizes the following optimization problem:

$$\boldsymbol{\theta}^* \leftarrow \arg\min_{\boldsymbol{\theta}} E(\boldsymbol{\theta}) \quad \text{with} \quad E(\boldsymbol{\theta}) = - \sum_{i=1}^n (1 − y^{(i)}) \ln(1 − \sigma^{(i)}) + (y^{(i)}) \ln(\sigma^{(i)})$$

with

$$ \boldsymbol{X} = \begin{pmatrix} 1 & x_1^{(1)} & \dots & x_p^{(1)} \\ \vdots & \vdots & \dots & \vdots \\ 1 & x_1^{(n)} & \dots & x_p^{(n)} \end{pmatrix} \quad \quad \boldsymbol{y} = \begin{pmatrix} y^{(1)} \\ \vdots \\ y^{(n)} \end{pmatrix} \quad \quad \boldsymbol{\theta} = \begin{pmatrix} \theta_0 \\ \vdots \\ \theta_p \end{pmatrix} $$

As there is no closed form for the optimal $\theta^*$, we will minimize $E$ using a gradient descent. Compute, either here or on a separate sheet of paper that you will give at the end of the exam, the gradient of $E$.

Recall:

$$ (\boldsymbol{A} + \boldsymbol{B})^T = \boldsymbol{A}^T + \boldsymbol{B}^T $$$$ (\boldsymbol{A} \boldsymbol{B})^T = \boldsymbol{B}^T \boldsymbol{A}^T $$\begin{align} \text{Associativity:} & \quad (\boldsymbol{A} \boldsymbol{B}) \boldsymbol{C} = \boldsymbol{A} (\boldsymbol{B} \boldsymbol{C}) = \boldsymbol{A} \boldsymbol{B} \boldsymbol{C} \\ \text{Non-commutativity:} & \quad \boldsymbol{A} \boldsymbol{B} \neq \boldsymbol{B} \boldsymbol{A} \\ \text{Distributivity:} & \quad \boldsymbol{A} (\boldsymbol{B} + \boldsymbol{C}) = \boldsymbol{A} \boldsymbol{B} + \boldsymbol{A} \boldsymbol{C} \\ & \quad (\boldsymbol{B} + \boldsymbol{C}) \boldsymbol{A} = \boldsymbol{B} \boldsymbol{A} + \boldsymbol{C} \boldsymbol{A} \end{align}

Differential identities for matrices:

\begin{align} d(\boldsymbol{A}) & = 0 \quad \quad \quad \text{if } \boldsymbol{A} \text{ is not function of } \boldsymbol{X} \\ d(a\boldsymbol{X}) & = a d \boldsymbol{X} \quad \quad \text{if } a \text{ is not function of } \boldsymbol{X} \\ d(\boldsymbol{X} +\boldsymbol{Y}) & = d \boldsymbol{X} + d \boldsymbol{Y} \\ d(\boldsymbol{XY}) & = (d \boldsymbol{X}) \boldsymbol{Y} + \boldsymbol{X} (d \boldsymbol{Y}) \\ d(\boldsymbol{X}^T) & = (d \boldsymbol{X})^T \\ \end{align}

Let's develop $E(\boldsymbol{\theta})$:

\begin{align} \nabla E(\boldsymbol{\theta}) & = \nabla (- \sum_{i=1}^n (1 − y^{(i)}) \ln(1 − \sigma^{(i)}) + (y^{(i)}) \ln(\sigma^{(i)})) \\ & = - \sum_{i=1}^n (1 − y^{(i)}) \frac{1}{1 - \sigma^{(i)}}(\sigma^{(i)} - 1)\sigma^{(i)} x^{(i)} + (y^{(i)}) \frac{1}{\sigma^{(i)}}\sigma^{(i)}(1-\sigma^{(i)}) x^{(i)} \\ & = - \sum_{i=1}^n x^{(i)} ((y^{(i)} - 1)\sigma^{(i)} + (y^{(i)}) (1-\sigma^{(i)})) \\ & = \sum_{i=1}^n x^{(i)}(\sigma^{(i)} - y^{(i)}) \\ & = \boldsymbol{X}(\boldsymbol{\sigma} - \boldsymbol{y})^T \end{align}

def grad_E(X, y, theta):
    """
        param X:     np.array of shape N x p
        param y:     np.array of shape N
        param theta: np.array of shape (p+1)
        return: np.array of shape (p+1)
    """
    ### BEGIN SOLUTION
    # First, we compute the value of sigma function for theta weights for each x in X
    # Second, we add a column of 1s for biases.
    # Finally, we compute X^T * (sigma - y).
    # Note here, that our matrices have a different orientation of the columns/rows than in the formula.
    # That's why the transpose is applied on X.
    pred = np.array([sigma(x, theta) for x in X])
    X = np.concatenate([np.ones((X.shape[0])).reshape(-1, 1), X], axis=1)
    return np.matmul(X.T, (pred - y))
    ### END SOLUTION

def grad_descent(X, y, eta=0.001, nb_max_step=1000, stopping_criterion=0.00001):
    """
        param x:                  np.array, input observations, shape N x p
        param y:                  np.array, input labels, shape N (filled with 0 and 1)
        param eta:                float, learning rate.
        param nb_max_step:        int, maximal number of steps done in the gradient descent.
        param stopping_criterion: float, stopping criterion on gradient process.
        
        return:                   np.array of shape p+1 (theta), list of float (evolution_of_loss)
        
        Remark: do not change the default values!
    """
    # Storing some useful values
    N, p = X.shape
    # List to store the evolution of E(theta) over steps
    evolution_of_loss = []
    # Store the current value of loss to stop gradient descent
    e = np.inf
    # Initialization of theta
    np.random.seed(seed=1)  # DO NOT change this
    theta = np.random.rand(p+1)
        
    ### Perform the gradient descent
    for t in range(nb_max_step):
        
        # Perform a gradient step
        #<-- to complete
        # new_e = ... #<-- to complete, store the loss according to the new parameter theta

        ### BEGIN SOLUTION
        # First, compute a gradient by calling grad_E function
        # Second, we update the theta parameters
        # by taking a step in the direction opposite to the gradient (descent).
        # We use eta as the learning rate.
        # Third, we store the new error in new_e that will later be compared with the previous error in e.
        grad = grad_E(X, y, theta)
        theta = theta - eta*grad
        new_e = E(X, y, theta)
        ### END SOLUTION
        
        # We stop the gradient descent if the variation in the loss function is small
        if np.abs(new_e - e) < stopping_criterion:
            break  # break allow to exit the for loop
        # otherwise, we just set e = new_e and store it
        else:
            e = new_e
            evolution_of_loss.append(e)
    
    return theta, evolution_of_loss
